# MinimalHttp.Net

MinimalHttp.Net is a simple and easy to use http client.

### NuGet

    Install-Package MinimalHttp.Net

### Features

- HttpClient GET, POST, HEAD, PUT, DELETE
- Certificate pinning
- Exception handling and input validation
- URLEncode & URLDecode
